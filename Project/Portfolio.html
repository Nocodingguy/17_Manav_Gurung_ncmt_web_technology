Projects

INsight

INsight is a frontend web application built using HTML, CSS, and JavaScript, designed to help users explore and navigate different parts of Kathmandu. The project features interactive maps, location highlights, and curated details about popular destinations, hotels, and essential services across the city. Created with first-time visitors in mind, INsight simplifies city exploration by offering easy access to places to visit, hotel booking options, and helpful travel information all within a clean and intuitive interface.
